# Font Files Needed

Place the following font files in this directory:

From **Manrope** (https://fonts.google.com/specimen/Manrope):
- Manrope-700.ttf (Bold)
- Manrope-600.ttf (SemiBold)

From **Inter** (https://fonts.google.com/specimen/Inter):
- Inter-400.ttf (Regular)
- Inter-500.ttf (Medium)
- Inter-600.ttf (SemiBold)

## How to get them:

1. Download from Google Fonts (links above)
2. Extract the ZIP files
3. Rename the font files to match the names above
4. Place them in this directory

## Alternative: Use GitHub CDN

You can also download pre-processed TTF files from:
- https://github.com/rsms/inter/releases (Inter fonts)
- https://github.com/sharanda/manrope/releases (Manrope fonts)
